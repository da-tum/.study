1. First Order: Non-Exact & Linear Non-Homogeneous DEs

8 Marks

A. Non-Exact DE → Finding Integrating Factor (I.F.)

Standard form: M dx + N dy = 0. Check if ∂M/∂y ≠ ∂N/∂x → non-exact. Find I.F. to make it exact.

1

Write in form M dx + N dy = 0. Compute ∂M/∂y and ∂N/∂x.

2

If ∂M/∂y ≠ ∂N/∂x, it is non-exact. Try I.F. formulas:

If (∂M/∂y - ∂N/∂x)/N = f(x) only → I.F. = e^(∫f(x)dx)
If (∂N/∂x - ∂M/∂y)/M = g(y) only → I.F. = e^(∫g(y)dy)
3

Multiply entire equation by I.F. → now exact. Solve as exact DE.

4

Integrate: F = ∫M dx (treat y as constant), then add terms from ∫N dy not already in F.

5

Solution: F(x, y) = C

Example (from Assignment 1, Q4): Solve (x²y² + y)dx + (2x³y − x)dy = 0
M = x²y² + y, N = 2x³y − x
∂M/∂y = 2x²y + 1, ∂N/∂x = 6x²y − 1 → Not exact
(∂M/∂y − ∂N/∂x)/N = (2x²y+1−6x²y+1)/(2x³y−x) = (−4x²y+2)/(x(2x²y−1)) = −2/x
I.F. = e^(∫−2/x dx) = e^(−2lnx) = x⁻²
Multiply through by x⁻² → solve as exact.

B. Linear Non-Homogeneous DE (1st Order)

Standard form: dy/dx + P(x)y = Q(x), where Q(x) ≠ 0 → Non-homogeneous.

1

Identify P(x) and Q(x).

2

Compute Integrating Factor: I.F. = e^(∫P dx)

3

Multiply both sides by I.F.: d/dx[y · I.F.] = Q · I.F.

4

Integrate both sides: y · I.F. = ∫Q · I.F. dx + C

5

Solve for y.

Example (from Question Bank): Solve dy/dx + (2/x)y = x²
P = 2/x, Q = x²
I.F. = e^(∫2/x dx) = e^(2lnx) = x²
y·x² = ∫x²·x² dx = ∫x⁴ dx = x⁵/5 + C
Answer: y = x³/5 + C/x²
Example (Assignment 1, Q5): Solve y′ − (1/x)y = x³, x > 0
P = −1/x, Q = x³
I.F. = e^(∫−1/x dx) = e^(−lnx) = 1/x
y·(1/x) = ∫x³·(1/x) dx = ∫x² dx = x³/3 + C
Answer: y = x⁴/3 + Cx

C. Bernoulli's Equation (Non-linear → Linearize)

Form: dy/dx + P(x)y = Q(x)yⁿ (n ≠ 0, 1). Used in Assignment 1 Q6–Q9.

1

Divide both sides by yⁿ.

2

Substitute v = y^(1−n), so dv/dx = (1−n)y^(−n) dy/dx.

3

Equation becomes linear in v: dv/dx + (1−n)P·v = (1−n)Q.

4

Solve using I.F. method, then substitute back.

Example (Q6): dy/dx + y/x = y² (n=2)
Divide by y²: y⁻²(dy/dx) + y⁻¹/x = 1
Let v = y⁻¹ → dv/dx = −y⁻²(dy/dx)
−dv/dx + v/x = 1 → dv/dx − v/x = −1
I.F. = e^(∫−1/x dx) = 1/x
v/x = ∫(−1)(1/x) dx = −lnx + C
v = −x·lnx + Cx → y = 1/v = 1/(Cx − x·lnx)

2. SVD (3×3) + Cubic Equation & Lagrange Multiplier

8 Marks

A. Singular Value Decomposition (SVD) — Step-by-Step

Goal: Decompose A = U Σ Vᵀ. For 3×3, eigenvalues lead to a cubic equation.

1

Compute AᵀA (an n×n symmetric matrix).

2

Find eigenvalues of AᵀA by solving det(AᵀA − λI) = 0. For 3×3 this gives a cubic equation in λ.

3

Singular values: σᵢ = √λᵢ (take positive square roots, arrange descending). These form the diagonal of Σ.

4

Find eigenvectors of AᵀA for each λᵢ → normalize → columns of V.

5

Find columns of U: uᵢ = (1/σᵢ)Avᵢ. Normalize if needed.

6

Write: A = U Σ Vᵀ

Example (Assignment 2, Q15) — A = [[1,1],[0,1],[−1,1]]
AᵀA = [[2,0],[0,3]]. Eigenvalues: λ₁=3, λ₂=2.
σ₁=√3, σ₂=√2. V = [[0,1],[1,0]] (eigenvectors of AᵀA).
u₁ = Av₁/σ₁ = [1/√3, 1/√3, 1/√3]ᵀ; u₂ = Av₂/σ₂ = [1/√2, 0, −1/√2]ᵀ
u₃ = orthogonal to u₁,u₂ via cross product.
Key point for 3×3: det(AᵀA − λI) = 0 expands to a cubic. Factor it as (λ − a)(λ − b)(λ − c) = 0. Use rational root theorem if needed.

B. Lagrange Multiplier Method

Optimize f(x,y) or f(x,y,z) subject to constraint g(x,y) = 0.

1

Set up Lagrangian: L = f(x,y) − λ·g(x,y)

2

Solve: ∂L/∂x = 0, ∂L/∂y = 0, ∂L/∂λ = 0

3

From the first two equations, express x and y in terms of λ.

4

Substitute into constraint to find λ, then find x, y.

5

Compute the optimal value f(x,y).

Example (Question Bank): Maximize f(x,y) = xy subject to x + y = 10
L = xy − λ(x + y − 10)
∂L/∂x = y − λ = 0 → y = λ
∂L/∂y = x − λ = 0 → x = λ → x = y
Constraint: x + x = 10 → x = 5, y = 5
Max f = 5×5 = 25 at (5,5)
Example (Vol. 2): Minimize f(x,y) = x² + y² subject to 2x + 3y = 13
∂L/∂x = 2x − 2λ = 0 → x = λ; ∂L/∂y = 2y − 3λ = 0 → y = 3λ/2
2λ + 3(3λ/2) = 13 → 6.5λ = 13 → λ = 2
x = 2, y = 3. Min f = 4 + 9 = 13

3. Eigenvalue–Eigenvector (3×3) + Linear Regression OR Gauss Methods

8 Marks

A. Eigenvalue and Eigenvector — 3×3 Matrix

1

Form characteristic equation: det(A − λI) = 0

2

Expand the 3×3 determinant → cubic in λ. Solve for λ₁, λ₂, λ₃.

3

For each λᵢ, solve (A − λᵢI)x = 0 via row reduction to find eigenvector.

For A = [[a,b,c],[d,e,f],[g,h,i]]: det(A − λI) = det([[a−λ, b, c], [d, e−λ, f], [g, h, i−λ]]) = 0 Expands to: −λ³ + (a+e+i)λ² − (...)λ + det(A) = 0
Tip — Shortcut for trace and det:
Sum of eigenvalues = trace(A) = a + e + i
Product of eigenvalues = det(A)
These help verify your roots.

B. Linear Regression (Least Squares)

Fit y = a + bx to n data points. Normal equations:

Σy = na + bΣx ... (1) Σxy = aΣx + bΣx² ... (2) Solve simultaneously for a and b.
1

Make a table: x, y, x², xy. Sum each column.

2

Plug sums into normal equations and solve for a and b.

3

Write the regression line y = a + bx.

Example (Vol. 2): Points (1,3), (2,5), (3,7)
n=3, Σx=6, Σy=15, Σx²=14, Σxy=34
Eq1: 15 = 3a + 6b → 5 = a + 2b
Eq2: 34 = 6a + 14b → 17 = 3a + 7b
Subtract (Eq1×3 from Eq2): 2 = b → a = 1
Answer: y = 1 + 2x

C. Gauss–Jacobi Method

Uses old values for all updates in one iteration.

1

Rearrange each equation for its diagonal variable.

2

For iteration k+1, use ALL values from iteration k (old).

3

Repeat until convergence.

Example (Question Bank): 4x + y = 5, x + 3y = 4. Start (0,0).
x = (5 − y)/4, y = (4 − x)/3
Iter 1: x₁ = (5 − 0)/4 = 1.25; y₁ = (4 − 0)/3 = 1.333 (uses OLD x=0)
Answer: x₁ = 1.25, y₁ = 1.333

D. Gauss–Seidel Method

Uses newest (just-computed) values immediately. Converges faster than Jacobi.

1

Rearrange each equation for its diagonal variable.

2

For each variable in the iteration, use the most recently computed values.

3

Repeat until convergence (usually ~3–4 iterations shown).

Example: Same system: x = (5 − y)/4, y = (4 − x)/3. Start (0,0).
Iter 1: x₁ = (5 − 0)/4 = 1.25; y₁ = (4 − 1.25)/3 = 0.916 (uses NEW x₁=1.25)
Key difference from Jacobi: y₁ uses x₁, not x₀
Pre-condition: Check diagonal dominance before applying either method. |aᵢᵢ| must be greater than sum of |aᵢⱼ| for j ≠ i.

4. Theory / Short Answer Questions

2 Marks Each

What is a Random Variable? Three Types?

A random variable is a variable whose value is determined by the outcome of a random experiment. It maps sample space outcomes to real numbers.

Three types:

  • Discrete: Takes countable values (e.g., number of heads in 5 tosses)
  • Continuous: Takes values in an interval (e.g., height, weight)
  • Mixed: Has both discrete and continuous components

What is Diagonally Dominant Matrix?

A square matrix A is diagonally dominant if, for every row i:

|aᵢᵢ| ≥ Σⱼ≠ᵢ |aᵢⱼ|

Strictly dominant when >, not ≥. Required for guaranteed convergence of Gauss-Jacobi and Gauss-Seidel methods.

Example: [[4,1],[2,5]] → |4|≥|1|, |5|≥|2| ✓

What is a Convex Set?

A set S is convex if for any two points x, y ∈ S, the line segment connecting them lies entirely within S.

For all x,y ∈ S and t ∈ [0,1]: tx + (1−t)y ∈ S

Examples: Circle, ellipse, line segment. Non-example: star shape, crescent.

Convex Function — 2 Applications

A function f is convex if the line segment between any two points on its graph lies above or on the graph:

f(tx + (1−t)y) ≤ tf(x) + (1−t)f(y)

Equivalently, f''(x) ≥ 0 everywhere.

Applications:

  • Optimization: any local min is a global min
  • Machine learning: loss functions (MSE, cross-entropy) are convex → gradient descent converges

Linear Regression — Physical Interpretation

Linear regression fits the best straight line y = a + bx through data points by minimizing the sum of squared errors (least squares).

Physical interpretation:

  • b (slope): Change in y per unit change in x. E.g., if x = temperature and y = pressure, b is how much pressure changes per 1°C rise.
  • a (intercept): Value of y when x = 0 (baseline/starting value).
  • It predicts the average relationship — the line passes through (x̄, ȳ).

What is a Matrix? Example

A matrix is a rectangular array of numbers arranged in rows and columns. An m×n matrix has m rows and n columns.

Example (2×3 matrix): A = [[1, 2, 3], [4, 5, 6]] Element aᵢⱼ is in row i, column j. a₁₂ = 2, a₂₃ = 6

Uses: Representing linear systems, transformations, storing data.

Central Limit Theorem (CLT)

If X₁, X₂, ..., Xₙ are i.i.d. random variables with mean μ and variance σ², then as n → ∞ (practically n ≥ 30), the sample mean X̄ approaches a normal distribution:

X̄ ~ N(μ, σ²/n) Standardized: Z = (X̄ − μ)/(σ/√n) ~ N(0,1)

Significance: Allows normal distribution methods even when the original population is not normal.

5. Unit 3 — Probability & Distributions

8 Marks

A. Continuous Random Variable: f(x) = kx² on [0, 5] — Find k, Mean, Variance, SD

(This is the exact question from your notes: "Random variable between 0 to ∫kx² & mean, variance, standard deviation")

1

Find k: Total probability = 1, so ∫₀⁵ kx² dx = 1

∫₀⁵ kx² dx = k[x³/3]₀⁵ = k(125/3) = 1 → k = 3/125
2

Mean: E[X] = ∫₀⁵ x·f(x) dx = ∫₀⁵ x·(3/125)x² dx

E[X] = (3/125)∫₀⁵ x³ dx = (3/125)[x⁴/4]₀⁵ = (3/125)(625/4) = 1875/500 = 15/4 = 3.75
3

E[X²]: E[X²] = ∫₀⁵ x²·(3/125)x² dx = (3/125)∫₀⁵ x⁴ dx

= (3/125)[x⁵/5]₀⁵ = (3/125)(3125/5) = (3/125)(625) = 15
4

Variance: Var(X) = E[X²] − (E[X])² = 15 − (3.75)² = 15 − 14.0625 = 0.9375

5

SD: σ = √0.9375 ≈ 0.968

Summary: k = 3/125, Mean = 3.75, Variance = 0.9375, SD ≈ 0.968

B. Binomial Distribution — Through PDF, Mean, Variance, Probability

Note from image: "No MGF, through PDF only — mean, variance, probability"

PMF: P(X = r) = ⁿCᵣ · pʳ · qⁿ⁻ʳ where q = 1 − p Mean: E[X] = np Variance: Var(X) = npq SD: σ = √(npq)
Example (Assignment 3, Q5): Binomial with mean = 2.4, variance = 1.44. Find P(X≥5) and P(1 < X ≤ 4).
np = 2.4, npq = 1.44 → q = 1.44/2.4 = 0.6 → p = 0.4
n = 2.4/0.4 = 6. So X ~ B(6, 0.4).
P(X=r) = ⁶Cᵣ(0.4)ʳ(0.6)^(6−r)
P(X≥5) = P(X=5)+P(X=6) = ⁶C₅(0.4)⁵(0.6)¹ + ⁶C₆(0.4)⁶
= 6(0.01024)(0.6) + 1(0.004096) = 0.036864 + 0.004096 = 0.04096
P(1<X≤4) = P(2)+P(3)+P(4) = 0.31104+0.27648+0.13824 = 0.72576
Example (Assignment 3, Q8): 10 coins, P(≥7 heads).
n=10, p=0.5, X~B(10,0.5)
P(X≥7) = P(7)+P(8)+P(9)+P(10)
= [¹⁰C₇ + ¹⁰C₈ + ¹⁰C₉ + ¹⁰C₁₀] / 2¹⁰
= [120 + 45 + 10 + 1] / 1024 = 176/1024 = 11/64 ≈ 0.172

C. Exponential Distribution — MGF, Mean, Variance + What is Exponential (Derivation)

Note from image: "MGF, mean & variance. What is exponential. Only derivation."

1

Definition: Exponential distribution models the time between events in a Poisson process.

PDF: f(x) = λe^(−λx), x ≥ 0, λ > 0 CDF: F(x) = 1 − e^(−λx) Mean: E[X] = 1/λ Var: Var(X) = 1/λ²
2

Derivation of Mean (by integration):

E[X] = ∫₀^∞ x · λe^(−λx) dx Let u = x, dv = λe^(−λx)dx → du = dx, v = −e^(−λx) E[X] = [−xe^(−λx)]₀^∞ + ∫₀^∞ e^(−λx) dx = 0 + [−(1/λ)e^(−λx)]₀^∞ = 0 − (−1/λ) = 1/λ ✓
3

MGF Derivation:

M_X(t) = E[e^(tX)] = ∫₀^∞ e^(tx) · λe^(−λx) dx = λ ∫₀^∞ e^(−(λ−t)x) dx (valid for t < λ)=λ · [1/(λ−t)]=λ/(λ−t)
4

Mean and Variance from MGF:

M'(t) = λ/(λ−t)² → M'(0) = 1/λ = Mean ✓ M''(t) = 2λ/(λ−t)³ → M''(0) = 2/λ² Var(X) = M''(0) − [M'(0)]² = 2/λ² − 1/λ² = 1/λ² ✓
Example (Question Bank): Bulb life ~ Exp, mean = 100 hrs. P(fails before 50 hrs)?
1/λ = 100 → λ = 0.01
P(X < 50) = 1 − e^(−0.01×50) = 1 − e^(−0.5) = 1 − 0.6065 = 0.3935 (39.35%)
