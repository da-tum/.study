🏆 Part A: Essential 2-Mark Theory Questions

1. Random Variable & Types

Function mapping sample space outcomes to real numbers. Discrete: countable values (e.g. coin tosses). Continuous: infinite interval values (e.g. exact bulb lifespan).

2. Diagonally Dominant Matrix

Square matrix where $|a_{ii}| \ge \sum_{j \ne i} |a_{ij}|$ in every row. Guarantees Jacobi/Seidel convergence. Example: $\begin{bmatrix} 4 & 1 \\ 2 & 5 \end{bmatrix}$.

3. Convex Set (2 Applications)

Line segment between any two set points lies entirely within the set. Applications: Linear Programming feasibility regions, Game Theory strategy spaces.

4. Convex Function (2 Applications)

Line segment between two graph points lies above or on the graph ($f''(x) \ge 0$). Applications: Global optimization proofs, Machine Learning MSE loss minimization.

5. Regression Interpretation

Fits $y=a+bx$. Slope $b$ is rate of change (pressure rise per $1^\circ\text{C}$ temperature increase). Intercept $a$ is baseline value at $x=0$.

6. Central Limit Theorem (CLT)

Sampling distribution of sample mean $\bar{X}$ of size $n$ approaches normal distribution $N(\mu, \sigma^2/n)$ for $n \ge 30$, regardless of population distribution.

1. First Order Non-Exact DE (Integrating Factor)

Method Explanation: Standard form $M dx + N dy = 0$. Check exactness condition $\frac{\partial M}{\partial y} \ne \frac{\partial N}{\partial x}$. If non-exact, compute Rule 1: $\frac{M_y - N_x}{N} = f(x)$ to find $I.F. = e^{\int f(x)dx}$, or Rule 2: $\frac{N_x - M_y}{M} = g(y)$ to find $I.F. = e^{\int g(y)dy}$. Multiply the entire equation by $I.F.$ to make it exact, then integrate $\int M dx + \int (\text{terms in } N \text{ without } x)dy = C$.

Mapped Question (Assignment 1 Q4): Solve $(x^2y^2+y)dx + (2x^3y-x)dy = 0$.

• $M = x^2y^2+y, \quad N = 2x^3y-x$. • $\frac{\partial M}{\partial y} = 2x^2y+1, \quad \frac{\partial N}{\partial x} = 6x^2y-1$. Not exact. • $\frac{M_y - N_x}{N} = \frac{(2x^2y+1) - (6x^2y-1)}{2x^3y-x} = \frac{-4x^2y+2}{x(2x^2y-1)} = -\frac{2}{x}$. • $I.F. = e^{\int -2/x \, dx} = e^{-2\ln x} = x^{-2} = \frac{1}{x^2}$. • Multiply DE by $\frac{1}{x^2}$: $\left(y^2 + \frac{y}{x^2}\right)dx + \left(2xy - \frac{1}{x}\right)dy = 0$. Now exact. • $\int \left(y^2 + \frac{y}{x^2}\right)dx = xy^2 - \frac{y}{x}$. Terms of $N$ without $x$: None.
Proper Solution: $$xy^2 - \frac{y}{x} = C$$

2. Second Order Non-Homogeneous DE

Method Explanation: Standard form $y'' + P y' + Q y = R(x)$. The general solution is $y = C.F. + P.I.$. To find Complementary Function (C.F.), solve auxiliary equation $m^2 + Pm + Q = 0$. For Particular Integral (P.I.), evaluate $\frac{1}{f(D)}R(x)$. If $R(x)=e^{kx}$, replace $D$ with $k$ provided $f(k) \ne 0$.

Mapped Question (Question Bank Vol 1): Find general solution of $y'' - 4y = e^{3x}$.

• Auxiliary equation: $m^2 - 4 = 0 \implies m = \pm 2$. • C.F. $= C_1 e^{2x} + C_2 e^{-2x}$. • P.I. $= \frac{1}{D^2 - 4}e^{3x}$. Replace $D=3$: $\frac{1}{3^2 - 4}e^{3x} = \frac{1}{9 - 4}e^{3x} = \frac{1}{5}e^{3x}$. • General solution $y = C.F. + P.I.$.
Proper Solution: $$y = C_1 e^{2x} + C_2 e^{-2x} + \frac{1}{5}e^{3x}$$

3. Singular Value Decomposition (SVD 3×3 / 2×3)

Method Explanation: Decompose matrix $A = U \Sigma V^T$. Compute symmetric matrix $A^T A$, find its eigenvalues $\lambda_i$ and singular values $\sigma_i = \sqrt{\lambda_i}$ (arranged in descending order along diagonal of $\Sigma$). Find orthonormal eigenvectors of $A^T A$ to form columns of $V$. Compute columns of $U$ using $u_i = \frac{1}{\sigma_i}A v_i$.

Mapped Question (Assignment 2 Q15): Find SVD of $A = \begin{bmatrix} 1 & 1 \\ 0 & 1 \\ -1 & 1 \end{bmatrix}$.

• $A^T A = \begin{bmatrix} 1 & 0 & -1 \\ 1 & 1 & 1 \end{bmatrix} \begin{bmatrix} 1 & 1 \\ 0 & 1 \\ -1 & 1 \end{bmatrix} = \begin{bmatrix} 2 & 0 \\ 0 & 3 \end{bmatrix}$. • Eigenvalues of $A^T A$: $\lambda_1=3, \lambda_2=2$. Singular values $\sigma_1=\sqrt{3}, \sigma_2=\sqrt{2}$. • $\Sigma = \begin{bmatrix} \sqrt{3} & 0 \\ 0 & \sqrt{2} \\ 0 & 0 \end{bmatrix}$. • Eigenvectors of $A^T A$: For $\lambda_1=3$, $v_1 = [0,1]^T$. For $\lambda_2=2$, $v_2 = [1,0]^T$. Thus $V = \begin{bmatrix} 0 & 1 \\ 1 & 0 \end{bmatrix}$. • Columns of $U$: $u_1 = \frac{1}{\sqrt{3}}A v_1 = \begin{bmatrix} 1/\sqrt{3} \\ 1/\sqrt{3} \\ 1/\sqrt{3} \end{bmatrix}$. $u_2 = \frac{1}{\sqrt{2}}A v_2 = \begin{bmatrix} 1/\sqrt{2} \\ 0 \\ -1/\sqrt{2} \end{bmatrix}$. $u_3 = \begin{bmatrix} 1/\sqrt{6} \\ -2/\sqrt{6} \\ 1/\sqrt{6} \end{bmatrix}$.
Proper Solution: $$A = \begin{bmatrix} \frac{1}{\sqrt{3}} & \frac{1}{\sqrt{2}} & \frac{1}{\sqrt{6}} \\ \frac{1}{\sqrt{3}} & 0 & -\frac{2}{\sqrt{6}} \\ \frac{1}{\sqrt{3}} & -\frac{1}{\sqrt{2}} & \frac{1}{\sqrt{6}} \end{bmatrix} \begin{bmatrix} \sqrt{3} & 0 \\ 0 & \sqrt{2} \\ 0 & 0 \end{bmatrix} \begin{bmatrix} 0 & 1 \\ 1 & 0 \end{bmatrix}$$

4. Lagrange Multiplier Constrained Extrema

Method Explanation: Optimize $f(x,y)$ subject to constraint $g(x,y)=0$. Define Lagrangian $L(x,y,\lambda) = f(x,y) - \lambda g(x,y)$. Compute partial derivatives $\frac{\partial L}{\partial x}, \frac{\partial L}{\partial y}, \frac{\partial L}{\partial \lambda}$ and set them equal to zero. Solve simultaneous equations to find critical points $(x,y)$ and evaluate $f(x,y)$.

Mapped Question (Question Bank Vol 1): Maximize $f(x,y) = xy$ subject to $x+y=10$.

• $L(x,y,\lambda) = xy - \lambda(x+y-10)$. • $\frac{\partial L}{\partial x} = y - \lambda = 0 \implies y = \lambda$. • $\frac{\partial L}{\partial y} = x - \lambda = 0 \implies x = \lambda$. Thus $x = y$. • $\frac{\partial L}{\partial \lambda} = -(x+y-10) = 0 \implies x+y = 10$. • Substitute $x=y$: $2x=10 \implies x=5, y=5$. Maximum value $f(5,5) = 5 \times 5 = 25$.
Proper Solution: $$\text{Max Value} = 25 \quad \text{at } (5,5)$$

5. Eigenvalue & Eigenvector (3×3 Characteristic Cubic)

Method Explanation: Characteristic equation $\det(A - \lambda I) = 0$. For a 3×3 matrix, this determinant expands into a cubic equation: $-\lambda^3 + \text{Tr}(A)\lambda^2 - M\lambda + \det(A) = 0$. Find roots $\lambda_1, \lambda_2, \lambda_3$. For each eigenvalue, find the null space of $(A - \lambda_i I)x = 0$ via Gaussian elimination to find the corresponding eigenvector.

Mapped Question (Assignment 2 Q11 matrix): Find characteristic equation for $A = \begin{bmatrix} 3 & 1 & 6 \\ -6 & 0 & -16 \\ 0 & 8 & -17 \end{bmatrix}$.

• $\text{Trace}(A) = 3 + 0 - 17 = -14$. • $\det(A) = 3(0 - (-128)) - 1(102 - 0) + 6(-48 - 0) = 3(128) - 102 - 288 = 384 - 390 = -6$. • Expand $\det(A - \lambda I) = 0 \implies -\lambda^3 - 14\lambda^2 - 47\lambda - 6 = 0$.
Proper Solution: $$\text{Characteristic Equation: } \lambda^3 + 14\lambda^2 + 47\lambda + 6 = 0$$

6. Linear Regression (Least Squares Normal Equations)

Method Explanation: Fit $y = a + bx$ to $n$ data points. Least squares minimizes squared vertical residuals $\sum (y_i - a - bx_i)^2$. Setting partial derivatives with respect to $a$ and $b$ to zero yields the normal equations: $\sum y = na + b\sum x$ and $\sum xy = a\sum x + b\sum x^2$. Prepare a table summing $x, y, x^2, xy$, and solve simultaneously for $a, b$.

Mapped Question (Question Bank Vol 2): Fit straight line $y=a+bx$ to $(1,3), (2,5), (3,7)$.

• Table sums ($n=3$): $\sum x = 1+2+3=6, \quad \sum y = 3+5+7=15$. • $\sum x^2 = 1^2+2^2+3^2 = 14, \quad \sum xy = 1(3)+2(5)+3(7) = 3+10+21 = 34$. • Normal Eq 1: $15 = 3a + 6b \implies 5 = a + 2b$. • Normal Eq 2: $34 = 6a + 14b \implies 17 = 3a + 7b$. • Multiply Eq 1 by 3: $15 = 3a + 6b$. Subtract from Eq 2: $2 = b$. • Put $b=2$ into $5=a+2(2) \implies a=1$. Line $y = 1 + 2x$.
Proper Solution: $$y = 1 + 2x$$

7. Gauss–Jacobi Iterative Solver

Method Explanation: Iterative method for linear systems $Ax=b$. Verify strict diagonal dominance $|a_{ii}| \ge \sum_{j \ne i} |a_{ij}|$. Rearrange each equation to isolate the diagonal variable. In iteration $k+1$, compute all variables using ONLY the values from the previous iteration $k$ (simultaneous updating).

Mapped Question (Assignment 2 Q17): Solve $20x+y-2z=17, \quad 3x+20y-z=-18, \quad 2x-3y+20z=25$. Start $(0,0,0)$.

• Rearrange: $x = (17 - y + 2z)/20, \quad y = (-18 - 3x + z)/20, \quad z = (25 - 2x + 3y)/20$. • Iter 1 (uses $x_0=0, y_0=0, z_0=0$): $x_1 = 17/20 = 0.85, \quad y_1 = -18/20 = -0.9, \quad z_1 = 25/20 = 1.25$. • Iter 2 (uses $x_1=0.85, y_1=-0.9, z_1=1.25$): $x_2 = (17 - (-0.9) + 2(1.25))/20 = 20.4/20 = 1.02$. $y_2 = (-18 - 3(0.85) + 1.25)/20 = -19.3/20 = -0.965$. $z_2 = (25 - 2(0.85) + 3(-0.9))/20 = 20.6/20 = 1.03$.
Proper Solution (Converges to): $$x = 1, \quad y = -1, \quad z = 1$$

8. Gauss–Seidel Iterative Solver

Method Explanation: Successive updating method for $Ax=b$. Check diagonal dominance. Rearrange each equation to isolate the diagonal variable. As soon as a variable is computed in iteration $k+1$, its new value is immediately used in the subsequent equations within the exact same iteration. Converges roughly twice as fast as Jacobi.

Mapped Question (Assignment 2 Q13): Solve $45x_1+2x_2+3x_3=58, \quad -3x_1+22x_2+2x_3=47, \quad 5x_1+x_2+20x_3=67$. Start $(0,0,0)$.

• Rearrange: $x_1 = (58 - 2x_2 - 3x_3)/45, \quad x_2 = (47 + 3x_1 - 2x_3)/22, \quad x_3 = (67 - 5x_1 - x_2)/20$. • Iter 1 for $x_1$: $x_1 = (58 - 0 - 0)/45 = 1.288$. • Iter 1 for $x_2$: Uses NEWEST $x_1=1.288$. $x_2 = (47 + 3(1.288) - 0)/22 = 50.864/22 = 2.312$. • Iter 1 for $x_3$: Uses NEWEST $x_1=1.288, x_2=2.312$. $x_3 = (67 - 5(1.288) - 2.312)/20 = 58.248/20 = 2.912$.
Proper Solution (Converges to): $$x_1 = 1, \quad x_2 = 2, \quad x_3 = 3$$

9. Continuous Random Variable $kx^2$ on $[0,5]$

Method Explanation: For continuous PDF $f(x)$, total probability $\int_{-\infty}^\infty f(x)dx = 1$. Mean $E[X] = \int x f(x)dx$. $E[X^2] = \int x^2 f(x)dx$. Variance $\text{Var}(X) = E[X^2] - (E[X])^2$. Standard Deviation $\sigma = \sqrt{\text{Var}(X)}$.

Mapped Question (Question Bank Vol 1): Given PDF $f(x)=kx^2$ for $0 \le x \le 5$, find $k$, Mean, Variance, SD.

• Find $k$: $\int_0^5 kx^2 dx = 1 \implies k[x^3/3]_0^5 = 1 \implies k(125/3) = 1 \implies k = 3/125$. • Mean $E[X] = \int_0^5 x(3/125)x^2 dx = (3/125)\int_0^5 x^3 dx = (3/125)[x^4/4]_0^5 = (3/125)(625/4) = 15/4 = 3.75$. • $E[X^2] = \int_0^5 x^2(3/125)x^2 dx = (3/125)\int_0^5 x^4 dx = (3/125)[x^5/5]_0^5 = (3/125)(3125/5) = 15$. • Variance $\text{Var}(X) = 15 - (3.75)^2 = 15 - 14.0625 = 0.9375$. • SD $\sigma = \sqrt{0.9375} \approx 0.9682$.
Proper Solution: $$k = \frac{3}{125}, \quad \text{Mean} = 3.75, \quad \text{Variance} = 0.9375, \quad \text{SD} \approx 0.968$$

10. Binomial Distribution (No MGF, PDF Only)

Method Explanation: Probability Mass Function $P(X=r) = \binom{n}{r}p^r q^{n-r}$ where $q=1-p$. Mean $\mu = np$, Variance $\sigma^2 = npq$. Used when an experiment consists of $n$ independent Bernoulli trials with binary success/failure outcomes and constant success probability $p$.

Mapped Question (Assignment 3 Q5): Binomial distribution with Mean $=2.4$, Variance $=1.44$. Find $P(X \ge 5)$ and $P(1 < X \le 4)$.

• $np = 2.4, \quad npq = 1.44 \implies q = 1.44/2.4 = 0.6 \implies p = 0.4, \quad n = 2.4/0.4 = 6$. Distribution $B(6, 0.4)$. • $P(X \ge 5) = P(X=5) + P(X=6) = \binom{6}{5}(0.4)^5(0.6)^1 + \binom{6}{6}(0.4)^6(0.6)^0$. $= 6(0.01024)(0.6) + 1(0.004096) = 0.036864 + 0.004096 = 0.04096$. • $P(1 < X \le 4) = P(2) + P(3) + P(4) = \binom{6}{2}(0.4)^2(0.6)^4 + \binom{6}{3}(0.4)^3(0.6)^3 + \binom{6}{4}(0.4)^4(0.6)^2$. $= 15(0.16)(0.1296) + 20(0.064)(0.216) + 15(0.0256)(0.36) = 0.31104 + 0.27648 + 0.13824 = 0.72576$.
Proper Solution: $$P(X \ge 5) = 0.04096, \quad P(1 < X \le 4) = 0.72576$$

11. Exponential Distribution Derivations

Method Explanation: Continuous PDF $f(x) = \lambda e^{-\lambda x}$ for $x \ge 0, \lambda > 0$. Models continuous waiting times between Poisson events. Memoryless property holds. To find Mean and Variance, integrate by parts or take derivatives of Moment Generating Function $M_X(t) = E[e^{tX}]$.

Mapped Question (Question Bank Vol 1 Derivations): Derive Mean, MGF, Variance, and solve bulb problem (Mean $=100$ hrs, fails before 50 hrs).

• Mean $E[X] = \int_0^\infty x \lambda e^{-\lambda x}dx$. Integration by parts ($u=x, dv=\lambda e^{-\lambda x}dx$): $E[X] = [-x e^{-\lambda x}]_0^\infty + \int_0^\infty e^{-\lambda x}dx = \frac{1}{\lambda}$. • MGF $M_X(t) = E[e^{tX}] = \int_0^\infty e^{tx}\lambda e^{-\lambda x}dx = \lambda\int_0^\infty e^{-(\lambda-t)x}dx = \frac{\lambda}{\lambda-t}$. • Variance: $M'(t) = \frac{\lambda}{(\lambda-t)^2} \implies M'(0)=\frac{1}{\lambda}$. $M''(t) = \frac{2\lambda}{(\lambda-t)^3} \implies M''(0)=\frac{2}{\lambda^2}$. $\text{Var}(X) = M''(0) - [M'(0)]^2 = \frac{2}{\lambda^2} - \frac{1}{\lambda^2} = \frac{1}{\lambda^2}$. • Bulb Problem: Mean $=1/\lambda=100 \implies \lambda=0.01$. $P(X < 50) = 1 - e^{-0.01(50)} = 1 - e^{-0.5} \approx 1 - 0.6065 = 0.3935$.
Proper Solution: $$\text{Mean} = \frac{1}{\lambda}, \quad M_X(t) = \frac{\lambda}{\lambda-t}, \quad \text{Variance} = \frac{1}{\lambda^2}, \quad P(X < 50) \approx 39.35\%$$

1. Order & Degree of DE

Q: Find order and degree of $(d^2y/dx^2)^3 + 4(dy/dx)^4 + y = \sin x$.
Solution: Highest order derivative is $d^2y/dx^2$ (Order = 2). Its power is 3 (Degree = 3). Answer: Order = 2, Degree = 3.

2. Linear vs Non-Linear DE

Linear: Dependent variable $y$ and derivatives appear in first power only, without products $y \cdot y'$. Non-Linear: Contains higher powers, products, or transcendental functions of $y$ (e.g. $\sin y$).

3. Exact DE Condition

An equation $M dx + N dy = 0$ is exact if it is the total differential $du=0$ of a function $u(x,y)$. Necessary and sufficient condition: $\frac{\partial M}{\partial y} = \frac{\partial N}{\partial x}$.

4. Integrating Factor (I.F.)

A non-zero function $\mu(x,y)$ by which a non-exact differential equation is multiplied to convert it into an exact differential equation.

5. IVP vs BVP

Initial Value Problem (IVP): Auxiliary conditions specified at a *single* point $x_0$. Boundary Value Problem (BVP): Conditions specified at *two or more distinct* boundary points.

6. C.F. vs P.I.

Complementary Function (C.F.): General solution of homogeneous equation $f(D)y=0$ containing arbitrary constants. Particular Integral (P.I.): Specific solution to $f(D)y=X(x)$ containing no arbitrary constants.

7. Solve Linear DE (2 Marks)

Q: Solve $dy/dx + (2/x)y = x^2$.
Solution: $P=2/x, Q=x^2$. $I.F. = e^{\int 2/x dx} = x^2$. $y x^2 = \int x^4 dx = x^5/5 + C$. Answer: $y = x^3/5 + C/x^2$.

8. Solve Exact DE (2 Marks)

Q: Solve $(2xy+3)dx + (x^2-1)dy = 0$.
Solution: $M_y=2x, N_x=2x$ (Exact). $\int (2xy+3)dx - \int 1 dy = C$. Answer: $x^2y + 3x - y = C$.

9. Solve Homogeneous DE

Q: Solve $(x^2+y^2)dx - 2xy dy = 0$.
Solution: Put $y=vx, dy/dx=v+x(dv/dx)$. Separate variables and integrate. Answer: $x^2 - y^2 = Cx$.

10. Solve Separable DE

Q: Solve $dy/dx = (x e^x)/y$.
Solution: $y dy = x e^x dx \implies y^2/2 = x e^x - e^x + C$. Answer: $y^2 = 2e^x(x-1) + K$.

11. Solve Bernoulli DE

Q: Solve $dy/dx + y = y^2$.
Solution: Put $v=y^{-1}$. DE becomes linear $dv/dx - v = -1$. $I.F.=e^{-x}$. Answer: $y = 1/(1+C e^x)$.

12. Solve IVP (2 Marks)

Q: Solve $dy/dx = -2xy, \quad y(0)=5$.
Solution: $dy/y = -2x dx \implies \ln|y| = -x^2+C \implies y=C e^{-x^2}$. $y(0)=5 \implies C=5$. Answer: $y = 5e^{-x^2}$.

13. Solve BVP (2 Marks)

Q: Solve $y''+4y=0, \quad y(0)=0, y(\pi/4)=2$.
Solution: $y = C_1 \cos(2x) + C_2 \sin(2x)$. $y(0)=0 \implies C_1=0$. $y(\pi/4)=2 \implies C_2=2$. Answer: $y = 2\sin(2x)$.

14. Solve 2nd Order Homogeneous

Q: Solve $d^2y/dx^2 - 5(dy/dx) + 6y = 0$.
Solution: Aux equation $m^2-5m+6=0 \implies (m-2)(m-3)=0 \implies m=2,3$. Answer: $y = C_1 e^{2x} + C_2 e^{3x}$.

15. Absolute, Relative, % Error

Absolute: $E_a = |X_{\text{true}} - X_{\text{approx}}|$. Relative: $E_r = E_a / |X_{\text{true}}|$. Percentage: $E_p = E_r \times 100\%$.

16. Bisection Method (2 Marks)

Q: Root of $x^3-x-1=0, [1,2]$, 2 iterations.
Solution: $f(1)=-1, f(2)=5$. $x_1=1.5, f(1.5)=0.875$. $x_2=(1+1.5)/2=1.25$. Answer: $x \approx 1.25$.

17. Secant Method (2 Marks)

Q: Root of $x^2-4=0, x_0=1, x_1=3$, 1 iter.
Solution: $f(1)=-3, f(3)=5$. $x_2 = 3 - 5\frac{3-1}{5-(-3)} = 3 - 10/8 = 1.75$. Answer: $x_2 = 1.75$.

18. Newton Raphson (2 Marks)

Q: Solve $x^3-2x-5=0, x_0=2$, 1 iter.
Solution: $f(2)=-1, f'(2)=10$. $x_1 = 2 - (-1)/10 = 2.1$. Answer: $x_1 = 2.1$.

19. Regula Falsi (2 Marks)

Q: Root of $x^2-3=0, [1,2]$, 1 iter.
Solution: $f(1)=-2, f(2)=1$. $x_1 = \frac{1(1)-2(-2)}{1-(-2)} = \frac{5}{3} \approx 1.666$. Answer: $x_1 \approx 1.666$.

20. LU Decomposition (2 Marks)

Q: LU for $A = [[2,3],[4,7]]$.
Solution: $U_{11}=2, U_{12}=3$. $L_{21}(2)=4 \implies L_{21}=2$. $2(3)+U_{22}=7 \implies U_{22}=1$. Answer: $L=[[1,0],[2,1]], U=[[2,3],[0,1]]$.

21. SVD Analytical Steps

Compute $A^T A$ and $AA^T$. Find eigenvalues and orthonormal eigenvectors of $A^T A$ for columns of $V$. Singular values $\sigma_i=\sqrt{\lambda_i}$ form $\Sigma$. Columns of $U$ are $u_i = (1/\sigma_i)A v_i$. Assemble $A=U\Sigma V^T$.

22. Gauss Seidel (2 Marks)

Q: Solve $4x+y=5, x+3y=4$, start (0,0).
Solution: $x_1=(5-0)/4=1.25$. Immediately use $x_1$: $y_1=(4-1.25)/3=0.916$. Answer: $x_1=1.25, y_1=0.916$.

23. Gauss Jacobi (2 Marks)

Q: Solve $4x+y=5, x+3y=4$, start (0,0).
Solution: $x_1=(5-0)/4=1.25$. Uses OLD $x_0=0$: $y_1=(4-0)/3=1.333$. Answer: $x_1=1.25, y_1=1.333$.

24. Cholesky Decomposition

Q: Cholesky for $A = [[4,2],[2,10]]$.
Solution: $L_{11}^2=4 \implies L_{11}=2$. $2L_{21}=2 \implies L_{21}=1$. $1^2+L_{22}^2=10 \implies L_{22}=3$. Answer: $L=[[2,0],[1,3]]$.

25. Forward Difference Table

Q: Forward difference table for $y=x^2, x=1,2,3$.
Solution: $y=[1,4,9]$. First diff $\Delta y = [4-1, 9-4] = [3,5]$. Second diff $\Delta^2 y = [5-3] = [2]$. Answer: $\Delta y=[3,5], \Delta^2 y=[2]$.

26. Discrete vs Continuous RV

Discrete: Takes countable distinct values (e.g. number of heads). Continuous: Takes uncountably infinite values within an interval (e.g. exact height or weight).

27. Mean, Var, SD Calculation

Q: $P(0)=0.2, P(1)=0.5, P(2)=0.3$.
Solution: Mean $=0(0.2)+1(0.5)+2(0.3)=1.1$. $E[X^2]=0+0.5+4(0.3)=1.7$. Var $=1.7-1.1^2=0.49$. SD $=\sqrt{0.49}=0.7$. Answer: Mean=1.1, Var=0.49, SD=0.7.

28. Bernoulli Distribution PMF

Q: Give PMF and Mean for Bernoulli distribution.
Solution: PMF $P(X=x)=p^x(1-p)^{1-x}$ for $x \in \{0,1\}$. Mean $E[X]=p$. Variance $=p(1-p)$.

29. Binomial Probability (2M)

Q: Fair coin tossed 4 times, exactly 2 heads.
Solution: $n=4, p=0.5, r=2$. $P(X=2) = \binom{4}{2}(0.5)^2(0.5)^2 = 6(0.0625) = 3/8 = 0.375$. Answer: $0.375$.

30. Poisson Probability (2M)

Setting partial derivatives $\frac{\partial S}{\partial a} = 0$ and $\frac{\partial S}{\partial b} = 0$ gives the fundamental Normal Equations: $$\sum y = na + b\sum x$$ $$\sum xy = a\sum x + b\sum x^2$$

Step 1: Construct the Summation Table

We are given $n=3$ data points. Let us calculate each necessary summation product:

Point $i$ $x_i$ $y_i$ $x_i^2$ $x_i y_i$
1 1 3 $1^2 = 1$ $1 \times 3 = 3$
2 2 5 $2^2 = 4$ $2 \times 5 = 10$
3 3 7 $3^2 = 9$ $3 \times 7 = 21$
Totals ($n=3$) $\sum x = 6$ $\sum y = 15$ $\sum x^2 = 14$ $\sum xy = 34$

Step 2: Substitute into Normal Equations

Substituting our totals into Normal Equation 1 ($\sum y = na + b\sum x$):

$$15 = 3a + 6b \implies a + 2b = 5 \quad \text{--- (Eq. 1)}$$

Substituting into Normal Equation 2 ($\sum xy = a\sum x + b\sum x^2$):

$$34 = 6a + 14b \implies 3a + 7b = 17 \quad \text{--- (Eq. 2)}$$

Step 3: Solve the Linear System

• Multiply Eq. 1 by 3 to match the $a$ coefficients: $$3(a + 2b) = 3(5) \implies 3a + 6b = 15$$ • Subtract this equation from Eq. 2: $$(3a + 7b) - (3a + 6b) = 17 - 15 \implies b = 2$$ • Substitute $b = 2$ back into Eq. 1: $$a + 2(2) = 5 \implies a + 4 = 5 \implies a = 1$$

• Therefore, our line equation is $y = 1 + 2x$.
• Verification Check: For $x=1$, model gives $y = 1 + 2(1) = 3$ (Matches data point 1 perfectly!).

Question 15 (8 Marks): Binomial Distribution (ELI5 Tutorial)

Q: A fair coin is tossed 4 times. Find the probability of obtaining exactly 2 heads. Explain the Binomial formula, parameters ($n, p, q$), combination math, Mean, and Variance in complete, easy-to-understand detail.

What is a Binomial Experiment? It is an experiment where: 1) There is a fixed number of repeated trials ($n$). 2) Each trial has exactly two outcomes: Success or Failure. 3) The probability of success ($p$) remains identical on every single trial. 4) All trials are completely independent.

1. Break Down the Problem Parameters

• $n = 4$: Total number of coin tosses.
• $p = 0.5$: Probability of getting Heads (Success) on a single fair coin toss.
• $q = 1 - p = 0.5$: Probability of getting Tails (Failure) on a single fair coin toss.
• $x = 2$: The exact number of successful Heads we wish to find.

2. The Binomial Formula & Combination Math

The probability of getting exactly $x$ successes in $n$ trials is given by:

$$P(X = x) = \binom{n}{x} p^x q^{n-x}$$

What does $\binom{n}{x}$ (read as "$n$ choose $x$") mean? It calculates the number of different ways we can arrange $x$ successes among $n$ total slots. The mathematical combination formula is:

$$\binom{n}{x} = \frac{n!}{x!(n-x)!}$$

Let us calculate $\binom{4}{2}$ step-by-step for our coin toss:

$$\binom{4}{2} = \frac{4!}{2!(4-2)!} = \frac{4 \times 3 \times 2 \times 1}{(2 \times 1) \times (2 \times 1)} = \frac{24}{2 \times 2} = \frac{24}{4} = 6$$ • ELI5 Meaning: There are exactly 6 different orderings where exactly 2 heads appear (HHTT, HTHT, HTTH, THHT, THTH, TTHH).

3. Calculate the Final Probability

Now substitute our values into the Binomial formula:

$$P(X = 2) = 6 \times (0.5)^2 \times (0.5)^{4-2}$$ $$P(X = 2) = 6 \times (0.5)^2 \times (0.5)^2 = 6 \times 0.25 \times 0.25 = 6 \times 0.0625 = 0.375$$ • Therefore, the probability of getting exactly 2 heads is $0.375$ (or $37.5\%$).

4. Calculate Mean and Variance

• Mean ($\mu$): Expected number of successes across $n$ trials. Formula is $\mu = n \times p$. $$\mu = 4 \times 0.5 = 2 \quad \text{(We expect 2 heads on average)}$$ • Variance ($\sigma^2$): Measures the spread or variation. Formula is $\sigma^2 = n \times p \times q$. $$\sigma^2 = 4 \times 0.5 \times 0.5 = 1$$
Proper Solution: $$P(2) = 0.375, \quad \text{Mean}=2, \quad \text{Var}=1$$
Question 16 (8 Marks): Exponential Derivations (Utmost Detail)

Q: Define the Exponential Distribution. Derive its Mean, Moment Generating Function (MGF), and Variance showing every single integration step and integration-by-parts formula in utmost detail.

Definition: The Exponential distribution models the waiting time between Poisson events. A continuous random variable $X$ has an exponential distribution with rate parameter $\lambda > 0$ if its Probability Density Function (PDF) is: $$f(x) = \begin{cases} \lambda e^{-\lambda x}, & x \ge 0 \\ 0, & x < 0 \end{cases}$$

Derivation 1: Verification of Total Probability equals 1

To verify $f(x)$ is a valid PDF, the integral over all space must equal 1:

$$\int_0^\infty f(x) dx = \int_0^\infty \lambda e^{-\lambda x} dx$$ • The antiderivative of $e^{-\lambda x}$ is $\frac{e^{-\lambda x}}{-\lambda}$. Evaluating limits from $0$ to $\infty$: $$\lambda \left[ \frac{e^{-\lambda x}}{-\lambda} \right]_0^\infty = -1 \left( \lim_{x \to \infty} e^{-\lambda x} - e^0 \right) = -1(0 - 1) = 1 \quad \text{(Verified!)}$$

Derivation 2: Mean $E[X]$ via Integration by Parts

The expected value or Mean is defined as $E[X] = \int_0^\infty x f(x) dx$:

$$E[X] = \int_0^\infty x \lambda e^{-\lambda x} dx$$ • We use Integration by Parts formula: $\int u \, dv = uv - \int v \, du$.
• Let $u = x \implies du = dx$.
• Let $dv = \lambda e^{-\lambda x} dx \implies v = \int \lambda e^{-\lambda x} dx = -e^{-\lambda x}$.

Substituting into the parts formula:

$$E[X] = \left[ -x e^{-\lambda x} \right]_0^\infty - \int_0^\infty (-e^{-\lambda x}) dx$$ • The boundary term $\left[ -x e^{-\lambda x} \right]_0^\infty$ evaluates to $0$ as $x \to \infty$ (exponential decay dominates polynomial growth) and $0$ at $x=0$. $$E[X] = 0 + \int_0^\infty e^{-\lambda x} dx = \left[ \frac{e^{-\lambda x}}{-\lambda} \right]_0^\infty = 0 - \left( -\frac{1}{\lambda} \right) = \frac{1}{\lambda}$$

Derivation 3: Moment Generating Function $M_X(t)$

The MGF is defined as $M_X(t) = E[e^{tx}] = \int_0^\infty e^{tx} f(x) dx$:

$$M_X(t) = \int_0^\infty e^{tx} \lambda e^{-\lambda x} dx = \lambda \int_0^\infty e^{tx - \lambda x} dx = \lambda \int_0^\infty e^{-(\lambda - t)x} dx$$ • For the integral to converge, we require $\lambda - t > 0 \implies t < \lambda$. $$M_X(t) = \lambda \left[ \frac{e^{-(\lambda - t)x}}{-(\lambda - t)} \right]_0^\infty = \frac{\lambda}{-(\lambda - t)} \left( \lim_{x \to \infty} e^{-(\lambda - t)x} - e^0 \right)$$ $$M_X(t) = \frac{\lambda}{-(\lambda - t)} (0 - 1) = \frac{\lambda}{\lambda - t} = \left( 1 - \frac{t}{\lambda} \right)^{-1}$$

Derivation 4: Second Moment $E[X^2]$ & Variance

To find Variance, we first find $E[X^2]$. We can differentiate the MGF twice with respect to $t$ and evaluate at $t=0$:

$$M_X'(t) = \frac{d}{dt}\left[ \lambda(\lambda - t)^{-1} \right] = \lambda(-1)(\lambda - t)^{-2}(-1) = \lambda(\lambda - t)^{-2}$$ $$M_X''(t) = \frac{d}{dt}\left[ \lambda(\lambda - t)^{-2} \right] = \lambda(-2)(\lambda - t)^{-3}(-1) = 2\lambda(\lambda - t)^{-3}$$

Evaluating at $t=0$ gives $E[X^2]$:

$$E[X^2] = M_X''(0) = 2\lambda(\lambda - 0)^{-3} = \frac{2\lambda}{\lambda^3} = \frac{2}{\lambda^2}$$

Finally, we compute Variance using $\text{Var}(X) = E[X^2] - (E[X])^2$:

$$\text{Var}(X) = \frac{2}{\lambda^2} - \left(\frac{1}{\lambda}\right)^2 = \frac{2}{\lambda^2} - \frac{1}{\lambda^2} = \frac{1}{\lambda^2}$$
Proper Solution: $$\text{Mean} = \frac{1}{\lambda}, \quad M_X(t) = \frac{\lambda}{\lambda - t}, \quad \text{Var}(X) = \frac{1}{\lambda^2}$$
